# Anant Tripathi Portfolio

A modern, highly interactive 3D developer portfolio built with React, Vite, Tailwind CSS, and Three.js (React Three Fiber). The site features a particle-driven 3D background scene, smooth animations, a responsive interface, and a built-in clean resume viewer.

## 🔗 Live Demo
Visit the live site: **[portfolio-3d-five-ruby.vercel.app](https://portfolio-3d-five-ruby.vercel.app)**

---

## ✨ Features

- **Interactive 3D Background:** Floating particle fields, orbiting rings, DNA helix streams, and an interactive wireframe globe that moves in response to mouse cursor movement (Three.js/Fiber).
- **Modern Dark UI:** Frosted glass panels (glassmorphism), neon gradient highlights, and a sleek dark theme.
- **Micro-Animations:** Fluid scrolling progress bar, interactive card overlays, and clean hover transitions powered by Framer Motion.
- **Technical Arsenal:** Categorized skills grid mapping programming languages, cybersecurity tools, and quantum research.
- **Education & Experience Timeline:** A visual representation of B.Tech CSE studies and Cybersecurity internship journey.
- **Resume Download:** Built-in downloadable, print-friendly, A4-formatted HTML resume.
- **Responsive Layout:** Optimized to look premium on desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack

### Frontend Core
- **React 19** - UI library
- **Vite** - Dev server & bundler
- **Tailwind CSS v4** - Styling framework
- **Framer Motion** - Interaction & entry animations

### 3D Graphics
- **Three.js** - 3D engine
- **React Three Fiber** - React wrapper for Three.js
- **React Three Drei** - Helper directives (Points, PointMaterial, Sphere, Grid)

### Deployment
- **Vercel** - High-performance hosting & continuous integration

## 🚀 Getting Started

To run this project locally, execute the following commands in your terminal:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```
   *The local server will start at `http://localhost:5173/`.*

3. **Build for production:**
   ```bash
   npm run build
   ```
